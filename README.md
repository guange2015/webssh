webssh with java implementation
===========================


WebSSH is a simple web project which support login linux server with explorer.

创意来自  `https://github.com/xsank/webssh.git`


一键运行：

`mvn clean && mvn tomcat7:run`

然后访问:

`http://127.0.0.1:9000`

没有然后了
